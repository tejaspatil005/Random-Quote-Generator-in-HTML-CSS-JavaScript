# Random-Quote-Generator-in-HTML-CSS-JavaScript
JavaScript  For Beginners
A simple web application that generates random quotes using JavaScript and an external API. The project demonstrates basic frontend development concepts including HTML, CSS, JavaScript, DOM manipulation, and API fetching.

Features

Displays a random quote and author

Fetches quotes from an external API

Button to generate a new quote

Simple and clean UI

Smooth button hover animation

Technologies Used

HTML5

CSS3

JavaScript (ES6)

Fetch API


How It Works

The application fetches quotes from the API:

https://dummyjson.com/quotes

The quotes are stored in a JavaScript array.

When the Generate button is clicked:

A random index is generated

A quote from the array is selected

The quote and author are displayed on the page.

Installation / Usage

Clone the repository or download the files.

Open the project folder.

Open index.html in your browser.

Click the Generate button to display random quotes.
